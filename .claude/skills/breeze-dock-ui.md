# BreezeDock Official UI/UX Design Skill (For Kimi 2.5 + Tailwind CSS v4)
## 核心风格：1:1 复刻 Linear 极简高级UI + Notion 舒适排版
## 配色系统：SoybeanUI 官方主色调（https://ui.soybeanjs.cn/）
## 核心指令（强制遵守）
请使用 frontend-design skill 构建所有页面结构、布局、视觉层级，**严格对标 Linear 应用的UI设计语言**，深度融合 Linear 极简风格 + Notion 舒适留白，100% 使用 SoybeanUI 官方配色，基于 shadcn/ui 组件体系 + Tailwind CSS v4 实现样式，确保界面和 Linear 视觉质感完全一致：极简、克制、高级、轻量化、长时使用舒适。同时实现 Linear 级流畅、无冗余的微交互。

---

## 全局技术约束（100% 强制）
1. **UI 组件库**：全部使用 shadcn/ui 官方组件（Button、Card、Sidebar、Separator、Input、Dialog、DropdownMenu、Tooltip、Badge）
2. **样式引擎**：全部使用 Tailwind CSS v4 原子类实现布局、动画、阴影、圆角、过渡、响应式适配
3. **禁止原生 CSS**：禁止手写 .css 文件、禁止手写 <style> 全局样式、禁止自定义样式污染
4. **不动业务逻辑**：只改 UI，不改任何功能、不增删逻辑、不修改数据/路由/事件
5. **强制配色**：禁止使用 shadcn/ui 默认配色，仅允许使用下方 SoybeanUI 官方颜色

---

## 全局风格约束（严格对标 Linear 设计规范）
1. **布局**：完全复刻 Linear 布局逻辑：侧边栏窄而紧凑、主内容区大量留白、层级清晰、无冗余装饰；侧边栏固定宽度 260px，主内容区 p-8，卡片间距 gap-4
2. **圆角**：统一使用 rounded-md（默认）/ rounded-lg（大卡片），Linear 极简克制圆角，拒绝大圆角
3. **阴影**：仅使用 shadow-none / shadow-sm，Linear 无厚重阴影，仅 hover 时极淡阴影
4. **字体**：现代无衬线字体，标题 font-medium，正文 font-normal，辅助文字 text-sm，完全对齐 Linear 字重层级
5. **线条**：极细、低对比度分隔线，仅用于区域划分，无多余边框
6. **交互**：所有动画 150–200ms、ease-in-out，Linear 级丝滑、无夸张动效
7. **组件变体**：优先使用 shadcn/ui ghost / secondary 变体，复刻 Linear 极简按钮风格

## 交互规范（严格对标 Linear 交互）
- 按钮 hover：柔和背景变化，无夸张动效，仅轻微提亮
- 按钮 active：轻微下沉 scale-98，无明显形变
- 卡片 hover：极淡阴影 + 背景微变，无抬升，保持 Linear 扁平感
- 切换/展开/折叠：height + opacity 平滑过渡 200ms，无抖动
- 输入框聚焦：柔和边框高亮，无刺眼效果
- 导航菜单：选中项背景微亮，hover 背景柔和变化，完全对齐 Linear 侧边栏交互
- 悬浮窗：丝滑过渡，无多余动画

## 页面视觉规范（严格对标 Linear 界面）
1. **侧边栏**：完全复刻 Linear 侧边栏样式：窄而紧凑、无多余装饰、选中项背景柔和、hover 反馈细腻；使用 shadcn/ui Sidebar 组件
2. **头部**：右侧必须包含 最小化、最大化/还原、关闭 三个窗口控制按钮，使用 ghost 变体，完全对齐 Linear 窗口按钮样式
3. **笔记列表**：复刻 Linear 列表质感：扁平、无卡片、hover 背景微变、极简线条
4. **编辑器**：无边框、无背景、留白充足、字体清爽（Notion 编辑器质感）
5. **标签组**：shadcn/ui Badge，SoybeanUI 主色、极简、圆角统一
6. **按钮**：shadcn/ui Button，优先 ghost / secondary 变体，Linear 极简风格
7. **图标**：线性简约 20x20，与文字对齐、层级统一，完全对齐 Linear 图标风格
